# spreadsheet-tools
O-spreadsheet deployment tooling


### dependencies
- Unix-based platform (linux/Mac Os)
- python 3.7+
- git 2.22+
- [GitHub CLI](https://cli.github.com/)

### Odoo configuration
This tool is used by internal employees that have access to the full development environment of Odoo.
Make sure that you have local repositories setup for Odoo Community and Odoo Enterprise with the respective
stable and development repositories.

### Installation
```bash
# From <root>/scripts
pip3 install -r requirements.txt
```

If you want to benefit from the autocompletion
```bash
# From <root>/scripts
pip3 install infi.docopt-completion

```
